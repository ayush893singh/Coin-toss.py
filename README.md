# Coin Toss Simulator

A simple Python project that simulates a single coin toss. The program randomly flips a virtual coin and displays either Heads or Tails.

---

# Features

- Flip a virtual coin
- Randomly display Heads or Tails
- Simple command-line interface
- Beginner-friendly Python project

---

# Technologies Used

- Python

---

# Concepts Used

This project uses:

- Variables
- User Input
- "random" Module
- "random.choice()"
- Output Formatting

---

# Installation

1. Install Python on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.

---

# How to Run

python main.py

---

# How It Works

1. Run the program.
2. Press Enter to flip the coin.
3. The program randomly selects either Heads or Tails.
4. The result is displayed on the screen.
